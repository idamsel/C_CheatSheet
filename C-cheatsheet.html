Dynamic Memory Allocation
Procedure in which the size of a data structure is changed during the
runntime.
malloc() - single large block of memory with the specified size. Returns
a pointer of type void.
calloc() - Dynamically allocate the specified number of blocks of memory
of the specified type. Initialize with default '0'.
realloc() - Dynamically change the memory allocation of a previously
allocated memory.
free() - Dynamically de-allocate the memory. Helps to reduce wastage of
memory by freeing it.
